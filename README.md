# PySpark Lakehouse Pipeline

Lakehouse-style data pipeline using PySpark and layered architecture.

## Project idea
This project simulates a data engineering pipeline using PySpark and a lakehouse-inspired structure with Bronze, Silver and Gold layers.

## Dataset
NYC Taxi Trips public dataset.

## Objective
Build a PySpark pipeline that:
- ingests raw trip data into a Bronze layer
- cleans and standardizes the data in a Silver layer
- creates analytical outputs in a Gold layer

## Planned layers
- Bronze: raw data ingestion
- Silver: cleaned and standardized data
- Gold: business-ready aggregated data

## Planned stack
- PySpark
- Python
- Parquet
- Jupyter Notebook or script-based pipeline
- GitHub

## Planned outputs
- partitioned data by layer
- cleaned datasets
- aggregated analytical tables
- project documentation

Project structure and planning phase.
